# Statistical Analysis of Study Hours Before and During Exams

## Overview
Performed statistical analysis to compare student study hours before and during examination periods.

## Technologies Used
- R
- ggplot2
- dplyr
- Statistical Testing

## Key Analysis
- Data preprocessing and cleaning
- Exploratory data analysis
- Comparison of study hour distributions
- Hypothesis testing and interpretation

## Outcome
Identified study pattern variations and statistical significance in study behavior during exams.
